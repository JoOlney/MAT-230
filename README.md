# MAT-230
Discrete Mathematics 

Assignments written in LaTeX using Codio and submitted as PDF files.

Topics Covered:

Logic & Proofs:
Translating English statements into predicate logic (∀, ∃, ∃!),
Applying De Morgan's laws to quantified expressions,
Proof by contrapositive, proof by contradiction, mathematical induction

Counting & Probability:
Multiplication rule, permutations, combinations (C(n,k)),
Inclusion-exclusion principle for sets and probabilities,
Complement counting (e.g., "at least one" problems),
Password counting with and without restrictions

Functions & Relations:
Function composition (f ∘ g) and inverses,
Relation properties: reflexive, anti-reflexive, symmetric, antisymmetric, transitive,
Equivalence relations and equivalence classes,

Graph Theory & Order:
Adjacency matrices and directed graphs,
Walk counting using matrix powers (A², A³),
Partial orders and Hasse diagrams — minimal/maximal elements, comparability

Reflection:

MAT 230 built the mathematical foundation that underlies most of computer science. Predicate logic and De Morgan's laws connect directly to how compilers evaluate conditionals. Counting and probability are the basis of algorithm analysis and cryptography. Relation properties — reflexivity, symmetry, transitivity — define the rules for data structures like ordered sets and databases.The most challenging topic was mathematical induction. The inductive step requires proving that a statement holds for n = m + 1 by assuming it holds for n = m, which feels circular until you see how the algebra closes the gap. Working through Bernoulli's inequality proof made the structure clear: the inductive hypothesis is a tool you use inside the proof, not a claim you are trying to prove.
